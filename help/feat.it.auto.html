Consenti agli utenti di creare caselle di posta
Quando questa funzione è abilitata, chiunque sarà in grado di utilizzare l'URL https://yourdomain.com:10000/virtualmin-signup/ per creare una nuova cassetta postale in questo server virtuale. Ciò consente di creare nuovi account e-mail senza l'intervento dell'amministratore, fino al limite consentito per le cassette postali per il server virtuale.
