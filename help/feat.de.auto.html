Ermöglichen Sie Benutzern das Erstellen von Postfächern
Wenn diese Funktion aktiviert ist, kann jeder die URL https://yourdomain.com:10000/virtualmin-signup/ , um ein neues Postfach auf diesem virtuellen Server zu erstellen. Auf diese Weise können neue E-Mail-Konten ohne Eingreifen des Administrators erstellt werden, bis die für den virtuellen Server zulässigen Postfächer begrenzt sind.
