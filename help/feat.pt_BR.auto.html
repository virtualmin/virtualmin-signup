Permitir que os usuários criem caixas de correio
Quando esse recurso está ativado, qualquer pessoa poderá usar o URL https://yourdomain.com:10000/virtualmin-signup/ para criar uma nova caixa de correio neste servidor virtual. Isso permite que novas contas de email sejam criadas sem a intervenção do administrador, até o limite de caixas de correio permitidas para o servidor virtual.
