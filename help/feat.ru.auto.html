Разрешить пользователям создавать почтовые ящики
Когда эта функция включена, любой сможет использовать URL-адрес https://yourdomain.com:10000/virtualmin-signup/ для создания нового почтового ящика на этом виртуальном сервере. Это позволяет создавать новые учетные записи электронной почты без вмешательства администратора, вплоть до ограничения на количество почтовых ящиков, разрешенных для виртуального сервера.
